# Zoryth AI Boss 🐉

**Zoryth AI Boss** is a Premium Boss Framework for Paper 1.21.x that enables server administrators to create unlimited, fully custom bosses purely through YAML configuration files.

## ✨ Features

- **Unlimited Custom Bosses**: Define bosses, their stats, drops, and logic in easy-to-use YAML configurations.
- **Advanced AI & Abilities**: Assign different targeting modes, dodge mechanics, and complex skill casting.
- **Dynamic Arenas**: Prevent players from escaping using enderpearls or chorus fruit, with customizable entry and exit lock delays.
- **Spawn Eggs**: Generate custom spawn eggs using `/zoryth giveegg`, now fully supporting `CustomModelData` in v1.5 for resource pack integration.
- **Customizable Boss Bars**: Dynamic boss bars with various colors and styles that animate on phase changes and show ability casts.
- **Extensive Integrations**:
  - **PlaceholderAPI**: For dynamic stats and information.
  - **Vault**: To provide economy-based rewards.
  - **WorldGuard**: Keep arenas safe and prevent bosses from spawning or destroying regions.
  - **Citizens**: Use NPC-based bosses and logic.

## 🚀 Commands

- `/zoryth create` — Create a new boss definition
- `/zoryth edit` — Edit a boss definition in-game
- `/zoryth spawn` — Spawn a boss
- `/zoryth giveegg` — Give a boss spawn egg
- `/zoryth list` — List all registered bosses
- `/zoryth kill` — Kill all active bosses
- `/zoryth reload` — Reload all configurations

*(All commands require `zoryth.admin` permission, or individual permissions like `zoryth.spawn`)*
